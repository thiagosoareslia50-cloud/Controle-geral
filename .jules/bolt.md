## 2024-05-18 - [Optimize buildMapData]
**Learning:** In V8/Node.js, writing a single-pass iteration using hardcoded, direct property assignments significantly outperforms consolidating map/list constructions via abstract helper functions (which invoke closures dynamically and iterate repeatedly). Extracting 16 dictionaries, 6 nested sets, and 9 array sets in a single pass over 5k elements reduced compute time from ~33ms to ~12ms. Standalone Node.js benching of `app.js` needs `global.window = {}` defined beforehand to parse correctly.
**Action:** When working with core state computation functions applied over potentially large datasets, prefer straightforward iteration logic over repeated O(N) traversals through dynamic array mapping methods, and ensure `global.window` is available in test scripts.
